# nekofetch
neofetch but with nekos

## Usage
```shell
Usage: nekofetch [options]
  Options:
    --help       shows this message
    -s|--sfw     load "safe for work" image
    -n|--nsfw    load "not safe for work" image
```

## Terminals
| Terminal  | Status      |
| :-------- | :---------- |
| kitty     | Ascii/Image |
| alacritty | Ascii       |
